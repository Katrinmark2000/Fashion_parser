# Fashion Article Aggregator

**Парсер и веб-приложение для сбора и отображения статей из модных журналов**

Проект, созданный в коллаборации двух разработчиц. Цель — автоматизировать сбор fashion-контента из популярных онлайн-изданий и отобразить его в удобном и визуально приятном интерфейсе.

---

## 🚀 О проекте

Fashion Article Aggregator — pet-проект, в котором бэкенд парсит статьи с разных модных сайтов, а фронтенд визуализирует полученные данные для пользователя. Это хорошая практика командной разработки, интеграции фронта и бэка через API и работы с реальными данными.

---

## Технологии

### 🔷 Фронтенд
- React (функциональные компоненты)
- TypeScript
- SCSS Module
- REST API

### 🔷 Бэкенд
- Go
- POSTGRES
- REST API
- Docker (контейнеризация бэкенда)

---

## Реализовано

- Вывод карточек статей с описанием и изображением
- Получение данных через REST API
- Локальное сохранение данных в LocalStorage
- Чистая структура проекта с разделением логики и компонентов
- Разворачивание Docker-контейнеров и настройка фронт части для корректной свяки с работой бека
